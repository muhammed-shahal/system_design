IP addresses allow devices to:

**Example:**

## üè∑ IP Types

### Public IP

A **public IP address** is assigned by your **Internet Service Provider (ISP)** and is used to identify your entire network on the internet.

- Same public IP is shared by all devices in your network
- Visible to the outside world

**Example:**  
IP address assigned to your router by the ISP.

---

### Private IP

A **private IP address** is assigned to devices inside your local network.

- Used for internal communication  
- Not reachable directly from the internet  
- Assigned by your router

**Example:**  
IP addresses given to your laptop, phone, smart TV, etc.

---

### Static IP

A **static IP address** does not change.

- Manually configured or permanently assigned
- More stable and reliable
- Usually costs more

**Common uses:**

- Servers  
- Remote access  
- Hosting services  
- Geo-location systems  

---

### Dynamic IP

A **dynamic IP address** changes over time.

- Assigned automatically using **DHCP**
- Most common type
- Cheaper and efficient
- Allows reuse of IPs

**Common uses:**

- Home internet connections  
- Mobile devices  
- Consumer networks
