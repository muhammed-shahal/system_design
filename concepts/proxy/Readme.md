# What is a Proxy?

A proxy is an intermediary server between a client and another server

Proxies help with security, caching, traffic control, and anonymity

# Forward Proxy

Sits between a client and the internet

can filter content, provide anonymity, and cache requests

Usecase: Privacy, Bypass geo restriction, Caching, Content Filtering

Example: VPN

# Reverse Proxy

Sits between users and beckend servers

client requests go through reverse proxy

used for load balancing, caching, security, and SSL termination

Usecase: Load balancing, Security and DDoS Protection. SSL termination

Example: Nginx, AWS ALB
