# IP Address

Internet Protool addresses are unique numerical labels assigned to devices on a network

They enable communication between different machines, servers, and services across the internet

Types of IPs

## IPv4

* This is the most widley used addressing system
* 32 - bit format (e.g. 192.168.1.1)
* ~4.3 billion addresses
* Challenges: Limited Ips, fragmentation, and security concerns

## IPv6

* Next generation IP standard
* 128 bit
* 340 undecillion addresses
* designed for IoT, mobiles, networks, and future scalability
* Better performance and security, improved routing

Catogories of IPs

## Public IPs

* Assigned by ISPs (Internet Service Provider)
* used to Communicate over the internet
* unique worldwide


## Private IPs

* used within local networks (LANs, enterprises, homes)
* Cannot be accsessed directly from the internet
